WHAT IS THIS GAME

Endless runner game inspired by Subway Surfers, built with Unity

[OVERVIEW]

An endless runner game inspired by the popular Subway Surfers, developed using the Unity game engine. The game features fast-paced action, obstacle avoidance, and progressively increasing difficulty to keep players engaged.

Built with Unity's powerful game development tools, this project demonstrates proficiency in game mechanics design, Unity's component system, physics, and player input handling for an addictive mobile gaming experience.

[KEY_FEATURES]

  • Endless Runner Mechanics: Infinite procedurally generated level design
  • Smooth Controls: Responsive swipe and tap controls for mobile
  • Obstacle System: Dynamic obstacle generation and avoidance mechanics
  • Progressive Difficulty: Gradually increasing speed and complexity
  • Score System: Distance-based scoring with power-ups
  • 3D Graphics: Unity's rendering pipeline for visual effects
  • Physics Integration: Unity physics for realistic movement

[GAME_MECHANICS]

Procedural Level Generation

Implemented a tile-based level generation system that creates infinite track segments on-the-fly, pooling and recycling objects for optimal performance on mobile devices.

Character Controller

Developed a smooth character controller with lane-switching mechanics, jump and slide actions, all responsive to touch input with fluid animations and state management.

Difficulty Scaling

Created a dynamic difficulty system that increases game speed, obstacle frequency, and complexity based on player progress, ensuring engaging gameplay at all skill levels.
