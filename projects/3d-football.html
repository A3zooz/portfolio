3D FOOTBALL PENALTY

Real-time 3D football penalty shootout built from scratch with C++ and OpenGL

[OVERVIEW]

A complete 3D football penalty shootout game featuring real-time rendering, physics simulation, and interactive gameplay. Built entirely from the ground up using C++ and OpenGL without using game engines, demonstrating deep understanding of computer graphics fundamentals.

The game features custom 3D model loading (.3DS format), advanced lighting systems, material properties, and physics-based collision detection, all implemented from scratch to create an immersive penalty shootout experience.

[KEY_FEATURES]

  • Real-time 3D Rendering: Complete graphics pipeline built with OpenGL
  • Custom Model Loader: Implementation of .3DS file format parser
  • Advanced Lighting: Multiple light sources with diffuse, specular, and ambient components
  • Material Systems: Realistic surface properties and texturing
  • Physics Simulation: Ball trajectory and collision detection
  • Interactive Gameplay: Player controls for penalty kicks
  • Camera System: Dynamic camera positioning and angles

[TECHNICAL_HIGHLIGHTS]

Custom 3D Model Loading

Implemented a complete .3DS file format parser from scratch, handling mesh data, vertex normals, texture coordinates, and hierarchical transformations without using external model loading libraries.

Lighting & Materials Pipeline

Built a comprehensive lighting system supporting multiple light types (directional, point, spot) with proper material properties including ambient, diffuse, specular, and shininess components for realistic rendering.

Physics-Based Ball Movement

Implemented realistic ball physics including gravity, velocity, spin, and collision detection with the goal and ground plane, creating authentic penalty kick mechanics.
