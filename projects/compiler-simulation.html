⚙️ COMPILER THEORY

Academic compiler construction project

[OVERVIEW]

A comprehensive Java project demonstrating fundamental compiler construction concepts including regular expressions, finite automata, and context-free grammars. This academic project implements key algorithms used in modern compiler design.

The project includes implementations for converting regular expressions to NFAs and NFAs to DFAs, performing various grammar transformations, and building an LL(1) parser for context-free grammars. All components are thoroughly tested to ensure correctness of the compiler theory implementations.

[KEY_FEATURES]

  • Regular Expressions to NFA: Thompson's construction algorithm implementation
  • NFA to DFA Conversion: Subset construction algorithm for deterministic automata
  • Grammar Transformations: Epsilon, unit, and left recursion elimination
  • FIRST & FOLLOW Sets: Computation algorithms for parsing table construction
  • LL(1) Parser: Top-down parser for context-free grammars
  • Finite Automata: Complete NFA and DFA implementations
  • Comprehensive Testing: Validated against compiler theory principles

[IMPLEMENTATION_DETAILS]

Automata Conversion Algorithms

Implemented Thompson's construction for regex to NFA conversion and subset construction for NFA to DFA transformation, both fundamental to lexical analysis in compilers.

Grammar Normalization

Developed algorithms for epsilon elimination, unit production removal, and left recursion elimination to transform grammars into parseable forms.

LL(1) Parser Construction

Built complete LL(1) parser including FIRST and FOLLOW set computation, parsing table generation, and predictive parsing algorithm implementation.
