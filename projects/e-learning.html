E-LEARNING PLATFORM

MERN stack course platform for Canadian Chamber of Commerce with progress tracking & certificate generation

[OVERVIEW]

A comprehensive e-learning platform built for the Canadian Chamber of Commerce using the MERN stack (MongoDB, Express, React, Node.js). Developed collaboratively in a team of 4, this platform enables users to attend pre-recorded courses online with full progress tracking.

The platform features an intuitive interface for content delivery, automatic certificate generation upon course completion, and responsive design for optimal user experience across all devices. The team worked closely with stakeholders to deliver project milestones and incorporate feedback.

[KEY_FEATURES]

  • Pre-recorded Courses: Access to professional pre-recorded course content
  • Progress Tracking: Real-time tracking of course completion and student progress
  • Certificate Generation: Automatic certificate issuance upon course completion
  • Responsive Design: React and CSS implementation for enhanced UX across devices
  • Team Collaboration: Delivered milestones based on stakeholder feedback
  • User Management: Secure authentication and user profile management

[TECH_STACK]

MongoDB
NoSQL database for flexible data storage
Express.js
Backend API framework
React
Frontend UI library
Node.js
Server-side JavaScript runtime
