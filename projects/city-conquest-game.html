CITY CONQUERING GAME

Turn-based strategy game with city management and world conquest mechanics

[OVERVIEW]

A comprehensive turn-based strategy game developed and maintained in Java, implementing core mechanics such as city management, army recruitment, and world conquest. The game features complex strategic gameplay where players manage resources, construct buildings, and command armies to conquer territories.

Built using object-oriented design principles with robust class hierarchies to model entities like players, cities, armies, and units. The game includes comprehensive exception handling to enforce gameplay rules and ensure balanced, error-free gameplay.

[KEY_FEATURES]

  • City Management: Build and upgrade cities with resource management
  • Army Recruitment: Create and manage diverse unit types and armies
  • Combat System: Strategic attack mechanics and siege warfare
  • Resource Management: Balance construction, upgrades, and military spending
  • Turn-Based Gameplay: Strategic planning with end-turn logic
  • Java Swing GUI: Interactive graphical interface for gameplay
  • Unit Testing: Comprehensive tests for recruitment, combat, and conquest

[TECH_STACK]

[CHALLENGES_&_SOLUTIONS]

Complex Game State Management

Implemented robust class hierarchies with clear separation of concerns to manage complex interactions between players, cities, armies, and units while maintaining code maintainability.

Balanced Gameplay

Designed custom exception handling and validation logic to enforce gameplay rules, ensuring fair and balanced strategic gameplay across all game mechanics.
