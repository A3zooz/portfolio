SPOKEN HABIT

Voice-powered habit tracker with hands-free management using LLMs

[OVERVIEW]

Spoken Habit is a cross-platform voice-powered habit tracker built with React Native, Expo Router, and Supabase. The app enables completely hands-free habit and task management through voice commands, making it perfect for busy individuals who want to maintain consistency without interrupting their workflow.

The app integrates real-time synchronization and authentication with Supabase, supporting secure multi-device usage with instant updates. AI-driven voice command parsing via Groq and Supabase Edge Functions allows users to add and complete habits/tasks simply by speaking.

[KEY_FEATURES]

  • Voice Recognition: AI-powered voice command parsing via Groq for natural interaction
  • Hands-free Operation: Complete voice control for adding and completing habits/tasks
  • Real-time Sync: Supabase backend with instant multi-device synchronization
  • Secure Authentication: Supabase authentication for multi-user support
  • Cross-platform: Built with React Native and Expo for iOS and Android
  • Edge Functions: Serverless AI processing with Supabase Edge Functions

[TECH_STACK]

React Native
Cross-platform mobile development
Supabase/PostgreSQL
Backend, authentication & real-time sync
Groq AI
Voice command parsing and understanding
Expo & Expo Router
Development platform & navigation

[CHALLENGES_&_SOLUTIONS]

Token Usage Optimization

Optimized API calls by sending transcripts of the user's voice using the speech recognition of the user's phone instead of sending raw audio data.

Offline Functionality

Developed a robust local storage system with background sync to ensure the app works seamlessly even without internet connectivity.
