IMAGE DENOISING

Bachelor Thesis (Grade A+): Deep learning autoencoder outperforming BM3D across multiple noise types

[OVERVIEW]

This project was completed as my Bachelor Thesis at The German University in Cairo, receiving a Grade A+. The research focused on using deep learning autoencoders for image denoising, achieving superior performance compared to the traditional BM3D algorithm.

Built with TensorFlow and Keras, the model features a symmetric encoder-decoder architecture with skip connections, enabling effective noise removal while preserving fine details across various noise types including Gaussian, Poisson, and salt-and-pepper noise.

[KEY_FEATURES]

  • Multiple Noise Types: Handles Gaussian, Poisson, and salt-and-pepper noise
  • Superior Performance: Outperforms traditional BM3D algorithm
  • Skip Connections: Preserves fine details and image structure
  • Transfer Learning: Pre-trained models available for quick deployment
  • Batch Processing: Efficient handling of large image datasets
