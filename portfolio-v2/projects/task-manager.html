TASK MANAGEMENT

Enterprise-grade task tracking system

[OVERVIEW]

A simple yet robust Task Manager web application developed using Java and Spring Boot. The application implements RESTful APIs for CRUD operations on tasks including add, update, delete, and fetch functionality.

The backend integrates PostgreSQL as the database with Spring JPA for seamless data persistence. The entire application is containerized using Docker and Docker Compose, making deployment easy and consistent across different environments.

[KEY_FEATURES]

  • RESTful APIs: Complete CRUD operations (add, update, delete, fetch) for tasks
  • Spring Boot: Java-based backend framework for robust development
  • PostgreSQL Integration: Reliable data persistence with relational database
  • Spring JPA: Object-relational mapping for seamless database operations
  • Docker & Docker Compose: Complete containerization for easy deployment
